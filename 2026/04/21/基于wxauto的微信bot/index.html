基于wxauto的微信bot

wxauto

无意间找到了这个项目，它可以识别和提取微信中的消息，还可以自动回复消息，算是一个简单的微信bot，还挺有意思的。
地址：https://wxauto.org/

python版本：python3.9-3.12。
系统：Windows10、11, Windows Server2016、2019、2022

安装

使用方法

使用时需要微信置于屏幕上可见的区域，并且打开需要监控的聊天消息界面。

开发实践

根据业务的需求，需要从微信群里截取坐标信息，然后我想直接自动显示在网页上。尝试用AI做一个程序来实现这方面的功能。
感觉推送端问题不大，只要使用push往指定端口推送消息就可以。但如何构建网站的处理机制，对我来说还是个困难问题，暂时还没发解决，还需要继续积累。

目录结构

wx_coordinate_monitor/
├── main.py # 主程序入口（GUI + 启动逻辑）
├── monitor.py # 微信监控核心逻辑
├── parser.py # 坐标解析（正则提取）
├── storage.py # 文件读写（JSON 持久化）
└── coordinates.json # 外部共享文件（网页可读）
