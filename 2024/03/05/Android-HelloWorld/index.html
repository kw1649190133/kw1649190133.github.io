Android-HelloWorld

前言

虽然并不是第一次接触安卓，但是之前在做安卓端部署YoloV8时碰了壁，所以现在回头恶补基本知识。众所周知，学习某个开发平台的第一个程序那肯定得是HelloWorld，今天就先做一个简单的程序出来给自己涨涨信心。

相关资料：

Android 官方网站：（https://developer.android.google.cn/?hl=zh-cn）

新建项目

AndroidStudio的新建项目里有不少县城的模板。现在知道E-hantai那些软件是怎么做出来的了，根本就是用的官方模板一点没改。

选择项目模板之后配置项目名、存放位置、SDK版本。

确认之后等待开发环境加载一段时间，AndroidStudio会给新项目创建一个新窗口。

这白花花的太晃眼了，顺便去设置里换一个主题。

Gradle问题

又遇到了和上次一样的问题：
project sync failed.Basic functionality(e.g.editing,debugging) will not work properly

然后会发现状态栏有个进程一直在下载gradle但下载不了。

去网站上直接下载gradle文件的压缩包，然后直接放在gradle目录下的乱码文件夹内。

然后点击try again，可以顺利运行了。

gradle下载站

(https://services.gradle.org/distributions/)

gradle问题总结

这下算是初步弄懂了gradle的问题了。就是每次对Android程序进行编译都需要gradle工具，而因为不同的程序需要各种不同版本的gradle，所以Android的设计就是需要的时候就从官网上下载一个到临时文件夹里。但因为服务器在灯塔国，中国用户因为墙的关系，日常被限制，就导致了连接超时。而程序员需要做的就是手动把对应版本的gradle压缩包下载下来，丢到用户名下那个临时文件夹里，帮愚蠢一根筋的IDE完成这个下载的过程。然后后面的过程就像窜稀一样流畅了，IDE会自己解压文件，运行构建程序，下载其他需要的组件。虽然解决完Gradle以后还是会报其他十几条问题，但是都不影响编译运行了。
但是像平时使用python时，把pip换个源就能解决的服务器连接问题，不知道为什么安卓解决不了，可能是因为AndroidStudio并没有开这样的设置吧。

调试运行

Android程序的运行必须在指定的环境下，可以使用现有的Android设备，或在电脑上运行虚拟环境。

在电脑上运行虚拟手机

在运行管理的界面除了能看到连接的手机设备，还能自己添加虚拟设备。按加号就可以进入虚拟设备创建页面，选择设备类型和规格（手机、平板、电视等），然后下一级选择系统版本。

到了选择系统界面如果左下角出现
”a system image must be selected“
就是对应的系统没有下载，直接点击左边的小白箭头下载。

在右边有个小提示可以看到各种版本的安卓使用分布情况，这下明白安卓为什么这么乱也就不稀奇了，这么多版本还能乱中有序也是挺神奇的。

然后选择运行设备，点击运行，就可以启动虚拟设备了。（有一种为了一碟醋包饺子的感觉，臃肿程度比C#更胜一筹。）

在手机上调试程序

  1. 把手机USB连上电脑。（要使用数据线）
  2. 进入开发者模式，打开USB调试开关。(像)现在的系统好像还支持WiFi调试，只要手机和正在同一个WIFI下即可。
    小米把开发者模式的开关藏在MIUI版本号里，在设置-全部参数里找到MIUI版本号狂点即可进入开发者模式，然后才能再更多设置里找到开发者选项。另外允许USB安装应用还必须要插入SMI卡，挺麻烦的。

调试运行之后APP就会被自动安装到手机上。

直接点开运行，非常流程丝滑。

目录解析

有个小细节，好像原来挺乱的项目文件夹怎么运行之后好像变了，其实是由项目文件夹切换到了安卓文件夹，在左上角可以切换回去。

三个主要文件：

  • activity：提供了用户与屏幕之间的互动，以便于用户进行操作，在里面实现主要的Java代码
  • activity_main.xml：布局文件（可能会没有），Android的UI界面显示的视图，所有的控件在这里设计。
  • AndroidManifest.xml：Android应用程序的清单文件,是整个Android应用程序的描述文件

至于安卓项目怎么布局的还是以后再分析吧，有点头大。

总结

虽然开发流程是成功跑起来了，但还是感觉安卓开发挺抽象的，前有服务器连不上，后有为了一个APP搞个虚拟手机，更别说这么多的安卓系统版本和硬件规格，要适配起来怕不是头都大了。

上次项目运行不起来应该也是gradle相关的环境或者解释器版本的问题。知道问题在哪之后解决起来应该就不会那么困难了。
